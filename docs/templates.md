# Templates & Standards / Plantillas y Estándares

## EN
Templates reduce cognitive load and enforce senior-level consistency.

Recommended standards:
- Repository template for playbooks
- Senior bilingual README
- Folder structure (docs/, examples/, assets/)
- Decision log (decisions.md)
- Quality rubric (scoring)

## ES
Los templates reducen carga mental y garantizan consistencia nivel senior.

Estándares recomendados:
- Template de repositorio para playbooks
- README senior bilingüe
- Estructura de carpetas (docs/, examples/, assets/)
- Registro de decisiones (decisions.md)
- Rúbrica de calidad (scoring)
