# Architecture / Arquitectura

## EN
This system is built around four layers, each with a clear purpose and audience.

Key rules:
- Each repository belongs to a single layer.
- Templates enforce consistency across layers.
- Layers should link to each other (profile → playbooks → products).

## ES
Este sistema se construye alrededor de cuatro capas, cada una con propósito y audiencia clara.

Reglas clave:
- Cada repositorio pertenece a una sola capa.
- Los templates aseguran consistencia entre capas.
- Las capas deben enlazarse entre sí (perfil → playbooks → productos).
