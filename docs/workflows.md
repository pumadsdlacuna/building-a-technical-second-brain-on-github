# Workflows / Flujos de trabajo (ChatGPT + GitHub)

## EN
Use ChatGPT as a system builder, not a content generator.

Workflow:
1. Define structure (architecture)
2. Create templates (standards)
3. Build Layer 1 repositories (playbooks)
4. Convert to Layer 2 products (execution)
5. Review with audits (continuous improvement)

## ES
Usa ChatGPT como constructor de sistemas, no solo generador de contenido.

Flujo:
1. Definir estructura (arquitectura)
2. Crear templates (estándares)
3. Construir repos de Capa 1 (playbooks)
4. Convertir a productos en Capa 2 (ejecución)
5. Revisar con auditorías (mejora continua)
