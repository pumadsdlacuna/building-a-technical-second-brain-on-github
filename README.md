# GitHub Personal OS

> A practical system to build a Technical Second Brain, professional brand, and product portfolio using GitHub.  
> Un sistema práctico para construir un Second Brain técnico, marca profesional y portafolio de productos usando GitHub.

---

## Overview / Descripción General

**EN**  
This repository documents a reusable approach to structure GitHub into layers:
- **Layer 0:** Profile & narrative (how you are perceived)
- **Layer 1:** Technical Second Brain (playbooks, patterns, decision criteria)
- **Layer 2:** Products & MVPs (execution and delivery)
- **Layer 3:** Strategic assets (frameworks, IP, open-core)

**ES**  
Este repositorio documenta un enfoque reutilizable para estructurar GitHub en capas:
- **Capa 0:** Perfil y narrativa (cómo te perciben)
- **Capa 1:** Second Brain técnico (playbooks, patrones, criterios de decisión)
- **Capa 2:** Productos y MVPs (ejecución y entrega)
- **Capa 3:** Activos estratégicos (frameworks, IP, open-core)
